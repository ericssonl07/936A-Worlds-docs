Compute the \doxylink{class_p_i_d}{PID} output for the given current value. 

calculate 
\begin{DoxyParams}{Parameters}
{\em current} & The current value. \\

{\em limit} & Whether to apply custom (non-\/traditional) \doxylink{class_p_i_d}{PID} calculations.\\

\end{DoxyParams}
This method computes the \doxylink{class_p_i_d}{PID} output for the given current value.

First, it calculates the error as the signed difference between the target and current values.

If limit is false, then the output is calculated using the traditional \doxylink{class_p_i_d}{PID} formula\+:

output = kp\+\_\+ \texorpdfstring{$\ast$}{*} error + ki\+\_\+ \texorpdfstring{$\ast$}{*} integral\+\_\+ + kd\+\_\+ \texorpdfstring{$\ast$}{*} derivative\+\_\+

If limit is true, then the output is calculated using the custom \doxylink{class_p_i_d}{PID} procedure\+:


\begin{DoxyItemize}
\item If \texorpdfstring{$\vert$}{|}error\texorpdfstring{$\vert$}{|} \texorpdfstring{$>$}{>} activation\+\_\+threshold, output = max\+\_\+value\+\_\+ \texorpdfstring{$\ast$}{*} sign(error).
\item If \texorpdfstring{$\vert$}{|}error\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} activation\+\_\+threshold, output = kp\+\_\+ \texorpdfstring{$\ast$}{*} error.
\item If \texorpdfstring{$\vert$}{|}error\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} integration\+\_\+threshold, output += ki\+\_\+ \texorpdfstring{$\ast$}{*} integral\+\_\+.
\item output += kd\+\_\+ \texorpdfstring{$\ast$}{*} derivative\+\_\+.
\item If \texorpdfstring{$\vert$}{|}output\texorpdfstring{$\vert$}{|} \texorpdfstring{$>$}{>} max\+\_\+value\+\_\+, output = max\+\_\+value\+\_\+ \texorpdfstring{$\ast$}{*} sign(output).
\item If \texorpdfstring{$\vert$}{|}output\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} min\+\_\+value\+\_\+, output = min\+\_\+value\+\_\+ \texorpdfstring{$\ast$}{*} sign(output).
\end{DoxyItemize}

Termination conditions are determined by the following\+:

arrived\+\_\+ = (\texorpdfstring{$\vert$}{|}error\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} tolerance\+\_\+ \&\& \texorpdfstring{$\vert$}{|}derivative\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} derivative\+\_\+threshold\+\_\+). \begin{Desc}
