This method calculates the intersection points between a line defined by two points and a circle defined by its center and radius.

It returns the parameter t in \mbox{[}0.\+0, 1.\+0\mbox{]} that corresponds to the progress along the line where the intersection occurs.

We parametrize the line by (1-\/t)P1 + t\+P2, 0\texorpdfstring{$<$}{<}=t\texorpdfstring{$<$}{<}=1, where P1 and P2 are the two points defining the line. \begin{DoxyNote}{Note}
Fundamentally, the equation reduces to a quadratic equation in t. The result can be classified depending on the discriminant\+:
\begin{DoxyItemize}
\item If the discriminant is negative, there is no intersection. We return -\/1.
\item If the discriminant is zero, there is one intersection point. We return t.
\item If the discriminant is positive, there are two intersection points. We return the larger t, which is the one that is closer to the second point P2.
\end{DoxyItemize}
\end{DoxyNote}
The function is static because it does not depend on the state of the \doxylink{class_pursuit}{Pursuit} object. \begin{DoxyReturn}{Returns}
The parameter t in \mbox{[}0.\+0, 1.\+0\mbox{]} that corresponds to the progress along the line where the intersection occurs, or -\/1 if there is no intersection.

It iterates forward through the path\textquotesingle{}s waypoints from the last found index to find the first point that is within the lookahead distance. At each step, it calls circle\+\_\+line\+\_\+intersection to check if the line from the robot\textquotesingle{}s position to the current point intersects with the circle defined by the lookahead distance.
