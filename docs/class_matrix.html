◆ row_operation()

void Matrix::row_operation ( int idx_row,
std::map< int, double > row_operations_map )
private

Perform a row operation on the matrix.

row_operation

Parameters
idx_rowIndex of the row to operate on
row_operations_mapMap of column indices and their corresponding multipliers
Note
Let r=idx_row, and row_operations_map be a set of tuples (i, k) where i is unique. Let the rows be r_{0}, ..., r_{n-1}.

Then the operation performed is:

r_{r} := r_{r} + k_{0} * r_{i_{0}} + k_{1} * r_{i_{1}} + ... + k_{m} * r_{i_{m}}

The operation is performed in-place.

◆ internal_matrix_memory

std::vector<std::vector<double> > Matrix::internal_matrix_memory
private

Internal storage for the matrix elements.

internal_matrix_memory

The matrix is stored as a 2D vector of doubles, where each element represents a value in the matrix.

internal_matrix_memory[i][j] represents the value at row i and column j.

Attention
Indexing starts at 0, so internal_matrix_memory[0][0] is the first element of the matrix. This contrasts with mathematical notation, where indexing starts at 1.
